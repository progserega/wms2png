Создание запроса на создание карты

Левый нижний угол квадрата нужной карты:

lat: lon:

Правый верхний угол квадрата нужной карты:

lat: lon:

Масштаб (от 1 до 17):

Например, для Владивостока вместе с русским островом будет:

Левый нижний угол квадрата нужной карты:
lat=42.9288
lon=131.7038
Правый верхний угол квадрата нужной карты:
lat=43.2312
lon=132.1219
Масштаб=17 (чтобы было видно наименования домов. Чем меньше масштаб - тем "легче" получаемая карта и меньше её детализация)

НЕ выбирайте большие территории и малый масштаб - карта будет такой большой, что она не загрузится в графический редактор для сведения слоёв.

Координаты можно взять на карте добавив в нужное место маркер и нажав по нему - в строке адреса будут координаты. Позже этот механизм будет переработан в виде простого выделения карты рамкой.
